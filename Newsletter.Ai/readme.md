# Camada AI

> Essa camada é responsável por conter as implementações de inteligência artificial da aplicação, como modelos de machine learning, processamento de linguagem natural e outras técnicas de IA. Ela é dependente da camada Core, mas não deve conter nenhuma lógica de negócio, apenas a implementação das interfaces definidas na camada Core para fornecer funcionalidades de IA à aplicação.
