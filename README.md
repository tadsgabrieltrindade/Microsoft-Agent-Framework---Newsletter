# Newsletter MAF

Projeto de aprendizado sobre **Microsoft Agent Framework** (MAF) com [balta.io](https://balta.io).

## 📋 Descrição

Uma aplicação de newsletter construída com o Microsoft Agent Framework para aprender sobre desenvolvimento de agentes inteligentes e orquestração de processos.

## 🏗️ Estrutura do Projeto

```
Newsletter/
├── Newsletter.Ai/          # Camada de IA e agentes
├── Newsletter.Api/         # API REST
├── Newsletter.Core/        # Lógica de negócio e entidades
├── Newsletter.Infra/       # Infra, banco de dados e integrações
└── Newsletter.slnx         # Solução
```

## 🎯 Objetivo

Explorar e aprender como usar o Microsoft Agent Framework para criar agentes que automatizam processos de newsletter.
